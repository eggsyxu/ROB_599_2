# ROB 498/599: Legged Robot Control

This repository contains example code and assignments for the course ROB 498/599: Legged Robot Control.

## Dependencies
- MATLAB >= R2018b
- Symbolic Math Toolbox
- Optimization Toolbox

## Setup Instructions

1. Install CasADi from https://web.casadi.org/get/
2. Run `setup.m` to add necessary paths.
3. Run `visualize_robot.m` to visualize the robot model.

## File Structure
- `robots/`: Contains example robot model files.
- `spatial_v2_extended/`: Contains the extended Spatial v2 library for dynamics computations.
- `setup.m`: Sets up the MATLAB environment.
- `visualize_robot.m`: Script to visualize the robot model.

> [!Note]
> If you didn't see any error messages in above steps, you are ready to go!
